# Clinica Médica
Projeto desenvolvido em PHP + Bootstrap 5 + Codeigniter v3.1.11

## Installation
Para instalar, necessário colocar a pasta /ci em xampp/htdocs

```bash
pip install foobar
```

## Usage

```python
import foobar

# returns 'words'
foobar.pluralize('word')

# returns 'geese'
foobar.pluralize('goose')

# returns 'phenomenon'
foobar.singularize('phenomena')
```
